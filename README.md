# PhD_project_adult_volunteers

This repository contains documentation and structured materials related to the adult volunteer component of my Ph.D. research project in Computer Science and Engineering.

## Project Context

The research investigates wrist-based Photoplethysmography (PPG) signal acquisition and its computational analysis in relation to cardiovascular and physiological indicators. The adult volunteer cohort forms part of the ethically approved data collection framework.

## Repository Contents

This repository includes:

- Informed Consent Forms (final formatted versions)
- Participant Information Sheet

## Ethical Compliance

Data collection procedures were conducted under approved ethical clearance. 
No real personal data is stored in this repository. Any participant list included here is synthetically generated for documentation and structural illustration purposes only.

## Scope

This repository specifically covers:

- Adult volunteers aged 22–60 years
- Documentation workflow
- Consent generation process
- Administrative and compliance materials

It does not contain:

- Clinical data
- Raw PPG signals
- Health records
- Identifiable participant information

## Purpose

The objective of this repository is to maintain transparent documentation of:

- Volunteer onboarding protocol
- Consent documentation structure
- Ethical research workflow
- Administrative reproducibility

This repository supports reproducible academic research practices and documentation integrity.
